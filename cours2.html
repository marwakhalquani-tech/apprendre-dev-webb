Introduction

CSS est utilisé pour styliser HTML et rendre les sites beaux.

👉 Sans CSS: site simple. Avec CSS: site professionnel.

Syntaxe CSS

selector {
  property: value;
}

👉 Exemple :

p {
  color: blue;
}

Extras CSS (Important 🔥)

Pseudo Classes

a:hover { color:red; }
a:active { color:green; }
input:focus { border:2px solid blue; }

👉 pseudo-class = état élément

Pseudo Elements

p::before { content:"🔥"; }
p::after { content:"✔"; }

👉 ajouter contenu sans HTML

Opacity

opacity:0.5;

👉 transparence élément

Transform

transform:scale(1.2);
transform:rotate(45deg);

👉 zoom / rotation

Overflow

overflow:hidden;
overflow:auto;

👉 contrôle contenu qui dépasse

Display

display:block;
display:inline;
display:none;

👉 block = ligne complète

CSS Variables

:root{
 --main-color:blue;
}

h1{
 color:var(--main-color);
}

👉 variables pour réutilisation

Object Fit

img{
 object-fit:cover;
}

👉 images propres sans distortion

Filter

filter:blur(5px);
filter:brightness(1.2);

👉 effets visuels

### 📄 Description des pages

- **index.html**  
  Page d'accueil du site. Elle contient une présentation générale du projet et introduit l'utilisateur au contenu du site.

- **page2.html**
  Page de choix. L'utilisateur sélectionne le langage qu'il veut apprendre (HTML, CSS, JavaScript). 

- **html.html**  
  Contient des leçons et exemples pour apprendre HTML.

- **css.html**  
  Contient des leçons et exemples pour apprendre CSS.

- **js.html**  
  Contient des leçons et exemples pour apprendre JavaScript.

- **practice0.html**  
  Page de pratique. Elle permet à l'utilisateur d'appliquer ce qu'il a appris à travers des quiz des site pour pratiquer toux ce qu'il apprends.

- **style.css**  
  Fichier de design pour styliser le site (couleurs, layout, responsive…).

- **script.js**  
  Fichier JavaScript pour ajouter des interactions (menu, animations…).

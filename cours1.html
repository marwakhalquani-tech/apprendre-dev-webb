Introduction à HTML

HTML signifie HyperText Markup Language. C’est le langage principal utilisé pour créer des sites web.

📌 À quoi sert HTML ?

  • Créer des titres, paragraphes, images
  • Structurer les pages web
  • Organiser le contenu

💡 Exemple simple :

<h1>Bienvenue sur mon site</h1>
<p>Ceci est mon premier paragraphe</p>

👉 Résultat : un titre + un texte affiché dans la page web.

Meta Tags (important pour le SEO 🔥)

Les meta tags donnent des informations sur la page au navigateur et aux moteurs de recherche.

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="mon site">

👉 Explication :

  • charset = caractères spéciaux
  • viewport = responsive design
  • description = SEO Google (important pour être bien référencé)

Structure de Base d’un Document HTML

Toute page HTML doit suivre une structure obligatoire.

📌 Pourquoi cette structure est-elle importante ?

  • Pour que le navigateur puisse lire et interpréter la page correctement.
  • Elle définit la base de notre page, incluant le contenu et les métadonnées.

💡 Exemple complet :

<!DOCTYPE html>
<html lang="fr">
  <head></head>
  <body></body>
</html>

👉 DOCTYPE : indique au navigateur qu'on utilise HTML5.

Les Balises HTML

Une balise HTML est utilisée pour structurer le contenu de la page web.

📌 Exemple :

<h1>Titre de la page</h1>
<p>Ceci est un paragraphe</p>
<div>Contenu dans un container</div>

💡 Explication :

  • <h1> : Balise de titre de premier niveau
  • <p> : Balise de paragraphe
  • <div> : Balise de container pour organiser le contenu

Les Attributs HTML

Les attributs sont utilisés pour donner des informations supplémentaires aux balises HTML.

📌 Exemple :

<a href="https://google.com" target="_blank">Google</a>

💡 Explication :

  • href = lien vers une page externe
  • target = où ouvrir le lien (_blank = nouvelle fenêtre)

Formatage de Texte

📌 Exemple :

<b>Texte en gras</b>
<strong>Texte important</strong>
<i>Texte en italique</i>
<u>Texte souligné</u>

  • strong : utilisé pour le texte important (SEO friendly)
  • b : texte en gras
  • i : texte en italique
  • u : texte souligné

Les Liens

Les liens permettent de naviguer entre les pages web.

📌 Exemple :

<a href="page2.html">Aller à la page 2</a>

Les liens sont très importants pour relier différentes pages d’un site ou pour accéder à d’autres ressources sur le web.

Les Images

Vous pouvez ajouter des images à votre page HTML pour un meilleur visuel.

📌 Exemple :

<img src="image.jpg" alt="description de l'image">

Le texte alternatif "alt" est utilisé pour l'accessibilité (au cas où l’image ne charge pas).

Les Listes

📌 Exemple :

<ul>
  <li>Élément 1</li>
  <li>Élément 2</li>
</ul>

<ol>
  <li>Premier</li>
  <li>Deuxième</li>
</ol>

ul = liste non ordonnée, ol = liste ordonnée

Les Tables (Tables HTML)

Les tables sont utilisées pour afficher des données sous forme de lignes et de colonnes.

📌 Exemple :

<table>
  <tr>
    <th>Nom</th>
    <th>Âge</th>
  </tr>
  <tr>
    <td>Ali</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Sara</td>
    <td>30</td>
  </tr>
</table>

💡 Explication :

  • <table> : balise principale pour définir une table.
  • <tr> : représente une ligne dans la table.
  • <th> : définit un en-tête de colonne.
  • <td> : définit une cellule de données.

👉 Résultat : une table affichant les noms et âges.

Les Formulaires HTML

Les formulaires permettent de collecter des informations auprès des utilisateurs.

📌 Exemple :

<form action="submit_form.php" method="POST">
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" placeholder="exemple@mail.com" required pattern="[a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$">

  <label for="phone">Numéro de téléphone:</label>
  <input type="tel" id="phone" name="phone" placeholder="06 12 34 56 78" required pattern="\d{2} \d{2} \d{2} \d{2} \d{2}">

  <label for="password">Mot de passe:</label>
  <input type="password" id="password" name="password" placeholder="********" required pattern=".{8,}" title="Le mot de passe doit contenir au moins 8 caractères">

  <button type="submit">Envoyer</button>
</form>

💡 Explication :

  • action : spécifie l'URL où les données seront envoyées.
  • method : définit la méthode HTTP utilisée (POST ou GET).
  • required : indique que le champ est obligatoire.
  • pattern : définit une expression régulière pour valider le format de la donnée.
  • title : affiche un message d'erreur si le format est incorrect.

📌 Patterns :

  • Email: [a-z0-9._%+-]+@[a-z0-9.-]+\.[a-z]{2,}$
  • Numéro de téléphone: \d{2} \d{2} \d{2} \d{2} \d{2} (format "06 12 34 56 78")
  • Mot de passe: .{8,} (doit contenir au moins 8 caractères)

Les Médias (Video, Audio, Iframe)

HTML permet d’intégrer des vidéos, de l'audio et des pages externes directement dans une page.

📌 Exemple :

<video controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.ogg" type="video/ogg">
  Votre navigateur ne prend pas en charge la vidéo.
</video>

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  Votre navigateur ne prend pas en charge l'audio.
</audio>

<iframe src="https://www.youtube.com" frameborder="0"></iframe>

💡 Explication :

  • <video> : balise pour intégrer une vidéo.
  • <audio> : balise pour intégrer un fichier audio.
  • <iframe> : balise pour intégrer une page web externe (comme YouTube).

HTML Sémantique

L'HTML sémantique permet d'améliorer l’accessibilité et le SEO en utilisant des balises spécifiques pour chaque section du contenu.

📌 Exemple :

<header>En-tête du site</header>
<nav>Navigation</nav>
<main>Contenu principal</main>
<section>Section de contenu</section>
<article>Article</article>
<footer>Pied de page</footer>

💡 Explication :

  • <header> : définit l'en-tête du site.
  • <nav> : définit la section de navigation.
  • <main> : définit le contenu principal de la page.
  • <section> : définit une section de contenu.
  • <article> : représente un article indépendant.
  • <footer> : définit le pied de page du site.

Accessibilité Web (Important pour tous 🔥)

Il est important que les sites web soient accessibles à tous, y compris aux personnes handicapées.

📌 Exemple d'accessibilité :

<img src="image.jpg" alt="Description de l'image">
<button aria-label="Fermer">X</button>

💡 Explication :

  • alt : texte alternatif pour les images (important pour les lecteurs d'écran).
  • aria-label : donne une description aux boutons pour les lecteurs d'écran.

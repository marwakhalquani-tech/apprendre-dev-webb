📌 Introduction à JavaScript

JavaScript est un langage de programmation qui permet de rendre les sites web interactifs.

👉 HTML = structure | CSS = design | JS = logique

JavaScript est exécuté dans le navigateur pour créer des effets dynamiques.

📌 Syntaxe JS

La syntaxe est la façon d'écrire le code JavaScript.

let name = "Ali";
console.log(name);

👉 console.log affiche un message dans la console du navigateur.

📌 Variables

Les variables servent à stocker des données.

var a = 10;
let b = 20;
const c = 30;

👉 let = modifiable | const = non modifiable

📌 Data Types

Les types de données en JavaScript :

  • String
  • Number
  • Boolean
  • Array
  • Object

📌 Operators

Les opérateurs permettent de faire des calculs et des comparaisons.

+ - * /
== ===
&& || !

📌 Conditions

Les conditions permettent de prendre des décisions dans le code.

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}

👉 Si la condition est vraie, le code s'exécute.

📌 Loops

Les boucles permettent de répéter du code.

for(let i = 0; i < 5; i++){
  console.log(i);
}

👉 Affiche les nombres de 0 à 4

📌 Functions

Les fonctions permettent de réutiliser du code.

function sayHello(){
  console.log("Hello World");
}

👉 Une fonction doit être appelée pour s'exécuter.

📌 Arrays

Un array est une liste de valeurs.

let arr = [1,2,3,4];
console.log(arr[0]);

📌 Objects

Les objets stockent des données sous forme clé/valeur.

let user = {
 name:"Ali",
 age:20
};

📌 DOM Manipulation

Le DOM permet de modifier le HTML avec JS.

document.getElementById("title").innerHTML = "Hello";

👉 On peut changer le contenu d’une page web.

📌 Events

Les événements sont des actions utilisateur.

button.onclick = function(){
 alert("Clicked");
};

📌 Advanced JS

setTimeout

setTimeout(() => {
 console.log("Hello");
}, 2000);

setInterval

setInterval(() => {
 console.log("Repeat");
}, 1000);

Arrow Function

const add = (a,b) => a + b;

Fetch API

fetch("api/data")
 .then(res => res.json())
 .then(data => console.log(data));
